Написать программу, выполняющую функционал калькулятора.



Программа считывает два числа с помощью функции scanf (scanf_s). (Как целые, так и числа с плавающей точкой; как положительные, так и отрицательные).
Программа считывает символ операции (Например +, -, *, /, % и тд).
Главные условия:
	- !!! НЕЛЬЗЯ ИСПОЛЬЗОВАТЬ ФУНКЦИИ ИЗ ЗАГОЛОВОЧНОГО ФАЙЛА math.h !!! (но можно проверять результаты собственных функций с функциями из math.h).
	- Все операции, должны быть вынесены в разные функции.
	- Если на вход переданы числа и операции, не удовлетворяющие требованиям, вывести сообщение об ошибке и завершить исполнение.
	- Вывести результат запрошенной операции в формате десятичной дроби.
	- Код должен быть максимально чистым, перменные должны называться в соответствии своему значению.
	- Некоторые операции требуют ограничений, например, операция вычисления факториала (!) работает только во множестве натуральных чисел (число должно быть целым и >= 0). Соответственно, необходимо сделать проверку на данное соответсвие.

Базовые операции: 
	//- Сложение (+). Входные - вещественные числа, результат - вещественное число.
	//- Вычитание (-). Входные - вещественные числа, результат - вещественное число.
	//- Умножение (*). Входные - вещественные числа, результат - вещественное число.
	//- Деление (/). Входные - вещественные числа, результат - вещественное число.
	//- Остаток от деления (%). Входные - вещественные числа, результат - вещественное число.
	//- Факториал (!). Входное - целое положительное число. Результат - целое положительное число (натуральное).
